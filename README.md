# Aurora Table Reservation Journey Map

A polished mobile web user journey map for a premium restaurant reservation experience. The design is inspired by Vercel-style minimalism and liquid glass UI: dark mode, translucent cards, soft gradients, layered depth, clean typography, and generous whitespace.

## What It Includes

- Five-step happy path: landing page, time selection, party size, confirmation, and success.
- Six parallel recovery paths: unavailable time slot, unsupported party size, SMS auth error, payment hold failure, network timeout, and fully booked waitlist conversion.
- Mobile-first iPhone-style UI mocks with realistic buttons, inputs, selectors, loading states, and concise elevated copy.

## Run Locally

```sh
npm start
```

Then open `http://localhost:5173`.

## Check Files

```sh
npm run check
```

This runs a small static validation script to confirm the required files and core journey content are present.
